Реализовать Web-приложение для управления подписками клиентов на газеты и журналы.

Требования:

Приложение должно уметь:

        1.1. Показывать список клиентов (общую информацию – ФИО и Адрес проживания) и его текущие подписки (наименование журнала/газеты).

        1.2. Добавлять подписку из списка существующих журналов/газет

        1.3. Удалять подписку из текущих подписок клиента

        1.4. Добавлять/удалять клиентов.

Наличие аутентификации обязательно

Список существующих подписок может быть фиксированным, то есть делать отдельную админку для подписок не обязательно

Использование микросервисной архитектуры

Приложение должно запускаться одной командой docker-compose
